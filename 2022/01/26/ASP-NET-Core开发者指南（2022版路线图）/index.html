ASP.NET Core开发者指南

2022年 ASP.NET Core 开发者指南：

在下面，您可以看到一个图，说明可以采用的路径以及要成为ASP.NET Core开发人员所想要学习的库。我将此图作为提示，给所有问我”作为ASP.NET Core开发人员，接下来应该学习什么？”的人。

免责声明

该指南的目的是为了给读者心有个大概的轮廓。如果你对接下来要学习的内容感到困惑时，这张路线图将指导你，而不是鼓励你选择时髦的东西。你应该逐渐理解为什么一种工具比另一种工具更适合某些场景，并且记住时髦和新颖的东西并不总是意味着最适合这个工作。

请给一个星星! :star:

如果你喜欢或正在使用这个项目进行学习或引用在你的解决方案中，请给它一个星星。谢谢!

路线图

资源

  1. 学习先决条件

    • C# 10
    • .NET 6
    • Entity Framework
    • Dapper
    • NHibernate
    • ASP.NET Core
    • SQL 基础知识

  2. 通用开发技能

    • 学习GIT 并且在 GitHub 中创建与分享项目
    • 掌握 HTTP(S) 协议与请求方法 (GET, POST, PUT, PATCH, DELETE, OPTIONS)
    • 不要害怕使用 Google，Google搜索技巧
    • 学习 dotnet CLI
    • 阅读一些关于算法和数据结构的书籍

  3. ASP.NET Core 基础

    • MVC
    • REST
    • Razor Pages
    • Razor Components
    • Middlewares
    • Filters & Attributes
    • Application Settings & Configurations
    • Authentication
    • Authorization
    • IdentityServer
    • Auth0
    • OIDC

  4. SOLID

    • 单一责任原则 (SRP)
    • 开放封闭原则 (OCP)
    • 里氏替换原则 (LSP)
    • 接口分离原则 (ISP)
    • 依赖倒置原则 (DIP)

  5. 依赖注入

    1. DI 容器
      • Microsoft.Extensions.DependencyInjection
      • AutoFac
      • Ninject
      • Castle Windsor
    • Simple Injector
    1. 生命周期
    2. Scrutor

  6. 数据库

    1. 关系型
      1. SQL Server
      2. PostgreSQL
      3. MariaDB
      4. MySQL
    2. 云数据库
      • CosmosDB
      • DynamoDB
    3. 搜索引擎
      • ElasticSearch
      • Solr
      • Sphinx
    4. NoSQL
      • Redis
      • MongoDB
      • Apache Cassandra
      • LiteDB
      • RavenDB
      • CouchDB

  7. 缓存

    1. 内存缓存
    2. 分布式缓存
      1. Redis
        1. StackExchange.Redis
        2. EasyCaching
      2. Memcached
    3. Entity Framework 二级缓存
      1. EFSecondLevelCache.Core
      2. EntityFrameworkCore.Cacheable

  8. 日志

    1. 日志框架
      • Serilog
      • NLog
    2. 日志管理系统
      • ELK Stack
      • Sentry.io
      • Loggly.com
      • Elmah.io

  9. API客户端和通信

    1. REST
      • OData
      • Sieve
    2. gRPC
    3. GraphQL
      • HotChocolate
      • GraphQL-dotnet

  10. 实时通信

  • SignalR
  • WebSockets
  1. 对象映射
  • AutoMapper
  • Mapster
  • ExpressMapper
  • AgileMapper
  1. 任务调度
  • Background Service
  • HangFire
  • Quartz
  • Coravel

  1. 测试

    1. 单元测试
      1. 框架
        • xUnit
        • NUnit
        • MSTest
      2. 模拟
        • Moq
        • NSubstitute
        • FakeItEasy
      3. 断言
        • FluentAssertion
        • Shouldly
    2. 集成测试
      • WebApplicationFactory
      • TestServer
    3. 行为测试
      • SpecFlow
      • BDDfy
      • LightBDD
    4. 端到端测试
      • Selenium
      • Puppeteer-Sharp

  2. 微服务

    1. 消息总线
      • RabbitMQ
      • Apache Kafka
      • ActiveMQ
      • Azure Service Bus
      • NetMQ
    2. 消息队列
      • MassTransit
      • NServiceBus
      • EasyNetQ
      • CAP
    3. API 网关
      • Ocelot
    4. 容器化
      • Docker
    5. 服务编排
      • Kubernetes
      • Docker Swarm
    6. 反向代理
      • YARP
    7. 其他
      • Orleans
      • Steeltoe
      • Dapr
      • Tye

  3. 持续集成与部署

    • Github Actions
    • Azure Pipelines
    • Travis CI
    • Jenkins
    • Circle CI
    • TeamCity

  4. 设计模式

    • CQRS
    • Decorator
    • Strategy
    • Builder

    • Facade

  5. 客户端库

    • Blazor

  6. 模板引擎
